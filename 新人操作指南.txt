============================================================
  抖B社群活动公告栏 — 新人操作指南（电脑端后台管理）
============================================================

这份指南教你如何在自己的电脑上启动后台管理网站。
不需要任何编程知识，跟着步骤做就行。

------------------------------------------------------------
第一步：安装 Node.js
------------------------------------------------------------

1. 打开浏览器，访问 https://nodejs.org
2. 点击左侧绿色的 "LTS" 按钮下载（20.x 或更高版本）
3. 双击下载好的 .msi 安装包
4. 一路点 "Next"（下一步），全部用默认选项
5. 最后点 "Install"，等待安装完成
6. 打开 PowerShell：
   - 按键盘上的 Win 键（Windows 图标键）
   - 输入 powershell
   - 回车
7. 在黑色窗口里输入：node --version
   如果显示 v20.x.x 之类的版本号，说明安装成功

------------------------------------------------------------
第二步：下载项目代码
------------------------------------------------------------

1. 创建一个文件夹放项目，比如在 D 盘建一个文件夹叫 "活动看板"
2. 打开你的文件管理器，进到这个文件夹
3. 打开浏览器，访问：
   https://github.com/SierraGao/xc-activityboard
4. 点击页面右上角的绿色 "<> Code" 按钮
5. 点 "Download ZIP"
6. 把下载的 ZIP 文件解压到你刚才建的文件夹里
   （解压后应该能看到 server.js、package.json、public 文件夹等）
7. 在 PowerShell 中进入这个文件夹：
   cd "你的文件夹路径"
   例如：cd D:\活动看板\xc-activityboard-master

------------------------------------------------------------
第三步：安装项目依赖
------------------------------------------------------------

在 PowerShell 中（确保在项目文件夹里），输入：

npm install

等待大约 1-2 分钟，出现 "added XX packages" 就成功了。

------------------------------------------------------------
第四步：创建密码文件
------------------------------------------------------------

在项目文件夹里创建一个新文件，命名为 .env
（注意：文件名就是一个点号加 env，没有其他后缀）

用记事本打开这个文件，写入以下内容后保存：

PORT=3000
ADMIN_PASSWORD=admin123
SESSION_SECRET=xb-2026-secret

⚠️ 建议把 admin123 改成你自己的密码。

------------------------------------------------------------
第五步：启动网站
------------------------------------------------------------

在 PowerShell 中（在项目文件夹里），输入：

node server.js

看到 "✅ 活动看板服务器已启动" 就成功了！

------------------------------------------------------------
第六步：访问后台管理
------------------------------------------------------------

打开浏览器，输入：

http://localhost:3000/admin

输入你设置的密码，登录后就可以管理活动了。

前台可以在浏览器打开：
http://localhost:3000

------------------------------------------------------------
每次开机后如何启动？
------------------------------------------------------------

每次重启电脑后需要重新启动服务器：
1. 打开 PowerShell
2. cd 到项目文件夹
3. node server.js

建议把项目文件夹固定到任务栏/开始菜单，方便快速打开。

------------------------------------------------------------
如何同步到公网？
------------------------------------------------------------

在后台管理页面点击 "同步到公网" 按钮。
1-5 分钟后，公网地址更新：
https://sierragao.github.io/xc-activityboard/

⚠️ 前提：你的电脑需要联网，且需要登录过 GitHub。
   如果同步失败，检查是否已安装 Git：
   https://git-scm.com/download/win
